Guide

Welcome to the PMP library guide! This guide provides detailed information about PMP.

The basics section contains the following pages:

  • Installation – detailed installation instructions
  • Overview – an overview of the library and its capabilities
  • Tutorial – learn the basics of working with PMP

The Algorithms section contains task-specific guides:

  • Decimation – reduce the number of triangles in a mesh
  • Remeshing – improve the shape and size of triangles
  • Subdivision – refine a mesh by subdividing elements
  • Smoothing – reduce noise and surface roughness
  • Hole filling – close holes in the mesh

The Advanced section covers special topics:

  • Interfacing with Eigen – interfacing with Eigen
  • Building JavaScript Apps – building JavaScript apps using emscripten

The Developers section contains guidelines for developers and contributors:

  • Contributing – instructions for submitting contributions
  • Coding Style – coding guidelines we follow around here

See the API Reference for detailed information on the classes and functions provided by PMP.
